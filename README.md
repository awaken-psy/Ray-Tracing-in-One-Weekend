# Ray-Tracing-in-One-Weekend

## 项目简介

基于《Ray Tracing in One Weekend》系列书籍的光线追踪器实现，用于学习和理解计算机图形学中的光线追踪技术。

## 构建说明

### 依赖

- C++ 编译器（支持 C++17 或更高版本）
- CMake 3.10+

### 编译

```bash
mkdir build
cd build
cmake ..
cmake --build .
```

### 运行

```bash
./RayTracing
```

输出图片将保存为 `output.ppm`
